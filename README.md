# Ez-Platformer
A Platformer (very easy)

***Note**: Slight use of ai to help me debug my code*

this is my 2nd game ever in godot game engine, which is an open source game engine, for hackclub horizons.
my game is a platformer type game, with spikes, checkpoints, coin collection and a fall system.
i hope you, the reviewer, like my game.

## Features:
- complete player movement (left, right and jumping system implemented)
- spike and falling system (which restarts the game when either action occurs)
- checkpoint system (which restarts at the checkpoint after death instead of start of the scene)
- coin collection systeam (to add a goal to the game and make it more fun)
- the gate (the winner scene thing)
- a parallax infinite generating background (its looped)
- a timer system (that tracks how long you take to beat the game)
- a pause menu (basic ui, which game doesnt have a pause menu?)
- winners scene (congratulates you for winning and asks if you want to replay)
- top notch music (i stole from other games lol)

## Controls:
- Use Left AND Right Arrow keys to move Left and Right respectively.
- Use the Space Bar to Jump.
- Use the Esc key to open the pause menu.

## Setup/Run instructions for the game
- go to itch.io, because my game is a browser embed
- click on the play button
- start the game by pressing left,right arrow keys or space bar on your keyboard to move
- anytime, during gameplay you can press Esc key to open the menu for the game
- with the menu, you can quit the game whenever you feel like it
